# Hyperframes Skill Notes

This folder keeps the agent-facing Hyperframes guidance used by this repository.

It is separated from the video examples so users can tell the difference between:

- reusable workflow and skill rules
- project-specific example scripts and narration
- local generated media

The notes here are not media assets. They document how agents should edit Hyperframes compositions safely: timed clips, `class="clip"`, registered timelines, muted video with separate narration audio, deterministic rendering, and the validation loop.
