## Style Prompt

Prompt Flow should feel like a lightweight desktop utility: fast, precise, quiet, and practical. The video uses a warm paper canvas, black editorial typography, restrained green accents, and real CLI demos as the main proof. Visuals should support the workflow instead of competing with it.

## Colors

- Canvas: `#F8F8F4`
- Ink: `#151716`
- Muted text: `#5F625C`
- Hairline: `#D8D9D2`
- Accent green: `#5D993E`
- Soft green surface: `#E7EEE2`

## Typography

- Display: `Arial Black`, `Microsoft YaHei`, sans-serif
- Body: `Microsoft YaHei`, system-ui, sans-serif
- Data / CLI labels: `Consolas`, `JetBrains Mono`, monospace

## Motion

- Fast utility motion: 0.2s to 0.55s entrances.
- Use push-slide and soft wipe transitions.
- Product footage stays dominant; text overlays should be short.
- Avoid dark tech style, neon gradients, large decorative logos, and generic slide-card grids.
